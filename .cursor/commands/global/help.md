# /global/help

Ajuda rápida dos comandos globais do Sistema Onion no Cursor.

## Objetivo
Fornecer um ponto de entrada enxuto para descobrir comandos e roteamento.

## Uso recomendado
1. Leia `AGENTS.md` para identidade e regras base.
2. Use os comandos de ajuda por domínio: `/engineer/help`, `/product/help`, `/docs/help`, `/git/help`.
3. Quando houver dúvida de especialista, delegue para `/onion`.
